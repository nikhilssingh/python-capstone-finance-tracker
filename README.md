# python-capstone-finance-tracker

This is a simple personal finance tracker that allows user to:
1) Add and categorize expenses
2) View all expenses by their category
3) Get a summary of total spending per category and total overall spending
4) Exit the program and receive a 'Goodbye' message

#It includes error handling as well

How to run the script:
1) Ensure you have Python installed
2) Save the script in a file (.py extension) 
3) Open a terminal in the same directory as the file 
4) Run it using python *insert name*.py
